このマニュアルは、制作進行シート（Google Spreadsheet）を使った日常オペレーションの進め方を、ポジション別にまとめたものです。

## 対象シート

制作進行シート（サブスクWEB制作）のメインタブを使用します。案件のステータスはこのシートで一元管理されます。

## シートの基本構造

シートは46列で構成されています。2行目が日本語ヘッダー、3行目以降がデータ行です。

主なグループは以下の通りです。

- **A〜F列**: 案件基本情報（顧客ID・パートナー名・契約プラン等）
- **G〜L列**: 体制（営業担当・ディレクター・Wix制作・納品希望日・未払い）
- **M〜P列**: ステータス（AI統合 / Wix統合 / Slack送信 / 対応完了）
- **Q〜W列**: 進行管理（ボール保持者・ポリッシュ・遅延管理）
- **X〜Z列**: 連絡方法（LINE / メール / アノテーション）
- **AA〜AC列**: 顧客情報（メール・LINE表示名・LINE ID）
- **AD列**: アポ録画メモ
- **AE〜AF列**: 営業共有事項（デモ作成前・デモ提出後）
- **AG〜AK列**: URL（ヒアリングシート・Gitリポジトリ・テストサイト・修正ツール・本番URL）
- **AL〜AQ列**: 完了必須チェック（リンク・フォーム通知・ファビコン・ドメイン・Google提出）
- **AR〜AT列**: 任意チェック（GA・GSC・Clarity — 未入力でも完了移動OK）
- **AU列**: 完了日（完了タブ移行の必須条件）

## ポジションの説明

| ポジション | 役割 |
| --- | --- |
| **パートナーサクセス（PS）** | 顧客からの連絡窓口。ステータスを変更し、適切な担当に連携する |
| **ポリッシュ** | AI制作案件のサイト制作チーム。キュー方式で上から順に対応し、制作から顧客への提出までを担う |
| **ディレクター** | 初回デモサイト完了時・スマホ最適完了時にデザイン品質のチェック・FBを行う |
| **Wixデザイナー** | Wix制作案件のサイト制作を行う担当者 |

## Q列「ボール保持者」の見方

Q列はM列・N列のステータスから**自動計算**されます。**手動で変更しないでください。** ステータスを変更すれば、Q列は自動的に更新されます。

| Q列の値 | 意味 |
| --- | --- |
| 営業 | 顧客対応・ヒアリングが必要な段階 |
| ポリッシュ | サイトの制作・修正が必要な段階 |
| ディレクター | デザインチェック・FBが必要な段階 |
| Wixデザイナー | Wix案件の制作・修正が必要な段階 |
| 顧客 | 顧客の回答・確認待ち |
| ― | 制作完了 |

## 色の見方

| 背景色 | 意味 |
| --- | --- |
| 赤系 | 遅延情報あり、もしくは未払い |
| 黄系 | 顧客待ちの状態 |
| 緑系 | 制作完了 |
| 白 | 通常（社内でボールを持っている） |

## 制作方式の見分け方

| 条件 | 制作方式 | ステータス列 |
| --- | --- | --- |
| M列（AI統合ステータス）に値がある | AI制作 | M列を見る |
| N列（Wix統合ステータス）に値がある | Wix制作 | N列を見る |

## タブ自動移動ルール

制作進行シートには「メイン」タブ以外に、「完了」「長期未返信」「未払い停止」「制作中解約」のタブがあります。案件は以下の条件で**GASにより自動移動**されます。

| 移動パターン | トリガー | 条件 |
| --- | --- | --- |
| メイン → **完了** | リアルタイム（セル編集時） | 以下の**必須7条件**が全て揃った時に確認ダイアログが表示され、「はい」で移動。①AK列（本番URL）入力済み ②AL列（リンクチェック）✅ ③AM列（フォーム通知）✅ ④AN列（ファビコン）✅ ⑤AO列（ドメイン）=「ドメイン接続完了」 ⑥AP列（Google提出）✅ ⑦AT列（完了日）に日付入力済み。**※AQ列（GA接続）・AR列（GSC接続）・AS列（Clarity接続）は未入力でもOK** |
| メイン → **長期未返信** | 日次バッチ（毎朝7時） | Q列（ボール保持者）=「顧客」かつT列（フェーズ期限日）から30日以上経過 |
| メイン → **制作中解約** | 日次バッチ（毎朝7時） | kintoneの受注ステータス（サブスクWEB制作）が「解約」 |
| メイン ↔ **未払い停止** | リアルタイム（セル編集時） | L列（is_unpaid）チェックボックスONで移動、OFFでメインに戻る（確認ダイアログ付き） |
| 各タブ → **メイン** | リアルタイム（セル編集時） | 長期未返信・未払い停止・制作中解約タブでM列またはN列のステータスが変更された場合に自動でメインに戻る |

> ⚠️ **手動で行をコピー&ペーストでタブ間移動しないでください。** 上記の自動移動は数式・フォーマット・バリデーションも含めて安全に移動します。手動コピーでは数式が壊れる可能性があります。
> 

## ディレクター自動振り分け

M列が「デモサイト制作中」に変更された際、I列（ディレクター）が空欄の場合、GASが自動でディレクターを割り当てます。

## 引継ぎシート自動記録

I列（ディレクター）またはJ列（Wix制作担当）が変更された際、別スプレッドシートの引継ぎシートに変更履歴（誰から誰に変更されたか・時期）が自動記録されます。
